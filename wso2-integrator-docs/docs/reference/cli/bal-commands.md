# bal Command Reference

## General Commands

| Command | Description |
|---------|-------------|
| `bal new` | Create a new package |
| `bal build` | Build the package |
| `bal run` | Run the package |
| `bal test` | Run tests |
| `bal clean` | Clean build artifacts |
| `bal doc` | Generate documentation |
| `bal pack` | Create distributable package |
| `bal push` | Push to Ballerina Central |
