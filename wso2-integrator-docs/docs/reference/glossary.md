# Glossary

| Term | Definition |
|------|-----------|
| **Automation** | A scheduled or manually triggered integration that runs without an external request |
| **Ballerina** | The cloud-native programming language powering WSO2 Integrator |
| **Connector** | A pre-built module for connecting to an external system |
| **Config.toml** | Environment-specific runtime configuration file |
| **Event Handler** | A reactive integration triggered by messages from brokers |
| **ICP** | Integration Control Plane — monitoring dashboard for running integrations |
| **Integration** | What you build in WSO2 Integrator — the unit of work |
| **Natural Function** | An LLM call treated as a typed function call |
| **Pro-code** | The Ballerina code editor view (as opposed to visual designer) |
| **Service** | An artifact that exposes endpoints over HTTP, GraphQL, gRPC, etc. |
| **Visual Designer** | The low-code drag-and-drop UI in VS Code |
