# FAQ

## General

### What is WSO2 Integrator?

WSO2 Integrator is a Ballerina-powered integration platform that combines low-code and pro-code development.

### Is WSO2 Integrator the same as WSO2 Micro Integrator?

No. WSO2 Integrator is a new product built on Ballerina, replacing the XML-based ESB model of MI. See the [migration guide](/docs/tutorials/migration/from-wso2-mi).

## Development

### Do I need to know Ballerina?

Not to get started. The visual designer handles most integration scenarios. Learning Ballerina unlocks advanced capabilities.

### Can I use existing Ballerina packages?

Yes. Any package on Ballerina Central can be imported and used.

## Deployment

### Where can I deploy?

Docker, Kubernetes, AWS, Azure, GCP, or WSO2 Devant (iPaaS). See [Deploy & Operate](/docs/deploy-operate).
